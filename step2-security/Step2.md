# step2 (step1 확장)
- Session + 쿠키 (보안 강화)
    - 비밀번호 암호화 (BCrypt)
    - 로그인 실패 횟수 제한 (3회)
    - 계정 잠금
    - 비밀번호 정책 (길이, 특수문자 - 유효성검사)
    - 아이디 정책 (유효성검사)
    - 아이디 찾기 (이메일 인증/일부 마스킹 처리)
    - 비밀번호 재설정 (이메일 인증)
    - 비밀번호 재사용 금지
    - login_history
    - password_history
    - remember-me
👉 보안 + 실시간 가능

회원가입
 → 이메일 검증
 → 비밀번호 정책 검사
 → BCrypt 암호화
 → password_history 저장

로그인
 → 실패 횟수 체크
 → 비밀번호 검증
 → 실패 시 카운트 증가
 → 성공 시 초기화
 → login_history 저장

비밀번호 변경
 → 과거 비밀번호 검사
 → 암호화 저장
 → history 저장

아이디 찾기
 → 이메일 인증
 → 마스킹 반환

remember-me
 → 쿠키 기반 자동 로그인
